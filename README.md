
# IA — Jeu de Nim (Python)

Implémentation d’un **jeu de Nim** avec une **IA** (stratégie optimale ou heuristique), pour illustrer la logique de jeu, les algorithmes et la prise de décision.

## Rappels (Nim)
Le Nim est un jeu à tours où l’on retire un certain nombre d’objets (allumettes, jetons, etc.) selon des règles fixées. Le joueur qui prend la dernière (ou au contraire qui évite la dernière) gagne, selon la variante.

## Objectifs du projet
- Implémenter les règles du jeu
- Coder une IA (stratégie gagnante si possible)
- Rendre le programme simple à lancer et à comprendre

## Prérequis
- Python 3.10+ recommandé
